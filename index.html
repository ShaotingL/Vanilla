Analysis Strategy

  1. Normalization：使用 DESeq-like median-ratio size factor。
  2. Rostellum signal：計算 S3_GR_vs_G 與 S4_GR_vs_G log2 fold change，並加入 replicate pairwise consistency。
  3. BGR/B support：用 BGR vs B 補充判斷 rostellum-related tissue signal。
  4. MADS-like co-expression：用 CR0040A 本地註解中的 MADS-like genes 作為 bait，計算候選基因相關性。
  5. Promoter CArG motif：掃描上游 2 kb CArG-like motif，作為可能被 MADS-box 直接調控的弱證據。
  6. Hold-out comparison：文章結果先保留不用於 ranking，等本分析完成後再做外部比較。

注意：目前是候選基因 prioritization，不是正式 DESeq2/edgeR 差異表現統計。

Biological Interpretation

Priority Tier 1: 調控與發育型候選

VANPL_A_00002t004950 與 VANPL_A_00010t001660 皆為 LOB domain-containing protein。LOB/LBD 類基因常與器官邊界、側生器官發育、局部生長調控相關，因此比單純代謝基因更適合作為「改變 rostellum 結構」的上游候選。

VANPL_A_00012t003490 與 VANPL_A_00001t028930 為 subtilisin-like protease，可能參與細胞外 peptide processing、組織 remodeling 或發育訊號調節。它們在我們重新 mapping 後的 CR0040A expression matrix 中具有 rostellum-associated signal，因此值得優先驗證。

Priority Tier 2: 生長支持與細胞壁候選

VANPL_A_00001t017650、VANPL_A_00004t004860、VANPL_A_00001t033180 涉及糖運輸或糖代謝控制。這類基因可能不是 rostellum fate 的主開關，但可能支援 rostellum 局部快速生長。

VANPL_A_00001t009440 與 VANPL_A_00003t002840 是 glycosidase/cell-wall related candidates，可作為細胞壁 remodeling 方向的驗證標的。

Recommended Validation Plan

  1. qPCR 第一輪：選 12-15 個 genes，涵蓋 LOB、subtilisin/CDR1-like、SWEET/MST、cell-wall enzymes 與 2-3 個 hypothetical top candidates。
  2. 樣本設計：優先重複 S3/S4 的 G vs GR 與 B vs BGR，比 S1/S2 更貼近 rostellum 形成與功能階段。
  3. 判斷標準：真正優先的候選應該在 GR 相對 G 顯著升高，且在 BGR 相對 B 也有同方向支持。
  4. 功能測試：若 qPCR 支持，先針對 LOB domain 與 subtilisin-like genes 做 transient silencing/VIGS 或局部 knockdown。
  5. 表型量測：量 rostellum 長度、寬度、位置、花粉接觸阻隔程度與花器官整體畸形程度，不只看授粉成功率。

Limitations

  • 目前 ranking 使用 effect size 與 consistency，不是 DESeq2/edgeR 的正式差異表現統計。
  • CArG motif 是 motif presence，不等於 MADS-box 實際 binding。
  • 文章品系與參考基因組版本不同；本階段不使用文章 HPP92/KAG crosswalk 或 WGCNA module 來排序候選基因。
  • 公共資料的 tissue/stage label 需要靠 qPCR 與形態樣本再次確認。
